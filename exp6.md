🔽 Step 1: Install VirtualBox
Go to official website:
Download VirtualBox
Download installer
Double-click .exe file
Click Next → Install → Finish

📌 VirtualBox installs like normal software

🆕 Step 2: Create Virtual Machine
Open VirtualBox
Click New
Enter:
Name: Ubuntu-VM
Type: Linux
Version: Ubuntu (64-bit)
Click Next

📌 VirtualBox provides a wizard to create VM easily

💾 Step 3: Allocate Memory (RAM)
Set RAM:
Minimum: 2 GB (2048 MB)
Click Next
💽 Step 4: Create Virtual Hard Disk
Select: Create Virtual Hard Disk Now
Click Create
Choose:
Type: VDI
Storage: Dynamically Allocated
Set size:
Minimum: 20 GB
Click Create

📌 VM needs storage space like real computer

📀 Step 5: Attach ISO File (OS Installation)
Select VM → Click Settings
Go to Storage
Click Empty → Choose Disk File
Select your ISO file (Ubuntu/Windows)
Click OK
▶️ Step 6: Start Virtual Machine
Select VM
Click Start

👉 OS installation screen will appear
👉 Follow installation steps (Next → Install → Finish)

📌 VM starts like a real computer

💻 Step 7: Use Virtual Machine
Login to OS
Install software
Use like normal system
⛔ Step 8: Shut Down VM
Click Shutdown inside VM
OR
Close → Select Power Off
